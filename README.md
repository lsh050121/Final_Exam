## 개요
### 1. 목적
스터디그룹을 **등록 및 검색**하여 원하는 스터디 그룹에 **참가**하고, 인원을 **모집**할 수 있는 프로그램입니다.    
보다 쉽게 스터디 그룹에 접근할 수 있도록 함으로써, **많은 이들의 교류를 이끌어 내는 것**이 목적입니다.

### 2. 대상
기본적으로 스터디 그룹에 관심있는 **청주대 학생들**을 대상으로합니다.    
추후, 청주대 이외의 사람들도 포함 시킬 예정입니다.
